Projeto: FreeBSD ARM64 Industrial Edge AI Platform

Olá, meu nome é Mauro Risonho de Paula Assumpção. 
Neste projeto desenvolvi uma plataforma de monitoramento baseada em edge computing utilizando FreeBSD ARM64, sistemas embarcados em C e análise de dados com inteligência artificial.

Inicialmente este projeto foi pensado para aplicações no segmento automotivo, onde veículos modernos produzem sinais contínuos de sensores como vibração, temperatura e corrente elétrica. Esses sinais podem indicar desgaste mecânico ou falhas iminentes.

A mesma arquitetura pode ser aplicada em ambientes industriais, onde máquinas também geram telemetria operacional que precisa ser analisada em tempo real.

Para resolver esse problema projetei uma arquitetura de edge computing baseada em FreeBSD ARM64 executando em QEMU, simulando um dispositivo embarcado responsável por coletar telemetria de sensores.

No dispositivo embarcado implementei serviços em C responsáveis pela aquisição de dados e transmissão de telemetria em JSON.

No backend desenvolvi um pipeline em Python para ingestão e análise de dados industriais, incluindo modelos de machine learning para detecção de anomalias.

Também implementei simulações de ataques usando uma abordagem de Red Team com apoio de IA, simulando manipulação de sensores e telemetria para validar se o sistema consegue detectar comportamentos adversariais.

Eu projetei a arquitetura completa do sistema, implementei os componentes embarcados, o pipeline de dados e os modelos de detecção. O resultado é uma plataforma que integra sistemas embarcados, ciência de dados e inteligência artificial para monitoramento industrial em edge computing.
